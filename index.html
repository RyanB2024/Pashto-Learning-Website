Welcome to Learn Pashto Today

A small, respectful, accessible site to help you learn Pashto — the main language of Afghanistan. Use the Practice area to start flashcards, quizzes, and listening exercises. This site is designed to work on desktop and mobile, and follows accessibility-first principles.
